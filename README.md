## How to build 
[Install gulp](https://github.com/leszek3737/Gulp-Config-Files)
install:
```
npm i -D jquery popper.js bootstrap@4.0.0-beta.3
```
Build dev
```
gulp build
```
Build prod
```
gulp build::prod
```
