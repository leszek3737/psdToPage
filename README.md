## How to build 
[Install gulp](https://github.com/leszek3737/Gulp-Config-Files)
and install:
```
npm i jquery popper.js bootstrap@4.0.0-beta.3
```
Build application in developer mode

```
gulp build
```
Build application in production mode

```
gulp build::prod
```
